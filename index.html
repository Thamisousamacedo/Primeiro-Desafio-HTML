Estrutura Básica do HTML

(voltar)

O HTML é composto pelo head e body. O head é o cabeçalho e tudo que estiver dentro do body é o que vai aparecer para o usuário. O head são as coisas que ele vai carregar antes de exibir para o usuário, e tem uma tag chamada title que é o título da página que a gente vai usar. Head vai ter algumas informações que são pré carregadas antes de exibir para o usuário. O body vai exibir para o usuário.

Tags e Atributos

(voltar)

A primeira Tag é a html que define o que é uma página html e tudo que estiver dentro dela é uma página html, o head pré carrega antes de aparecer para o usuário e tudo que estiver entre o body é o que vai aparecer para o usuário.

Existem vários outras Tags e cada uma com sua função, no VS Code quando digitamos a Tag ele aparece a entrada e saída, ex: . Alguns exemplos de Tags:i a letra fica em itálico e strong fica em negrito.

Algumas Tags não tem a barra de fechar, uma delas é a Tag input ao utilizar ela abre um campo para digitar, a Tag img também não tem a barra para fechar. Algumas vamos precisar fechar e outras não.

Atributos são propriedades que uma tag pode ter, a Tag idé um identificador que no Javascript define qual elemento eu quero usar.

Tag style, que faz parte dos comandos do CSS que é um sistema de formatar o html, são comandos específicos para deixar a página mais bonito alterando a cor e tamanho da fonte.

Outro atributo voltado para o CSS é o class, tudo que estiver dentro dele fica com uma cor padronizada, tudo que for um título principal ele vai ficar padronizado. O input que é usado para abrir um campo para digitar um texto muito usado em formulários, e ele tem o atributo type que é específico do input, ele pode receber text ou numb. No campo numb aparece as setas e só posso digitar números.

Outro atributo é o color usado para alterar a cor.

Tem também um atributo que é bem específico é da Tag img é o src, o img precisa do src é o espaço para digitar o endereço de uma imagem, que pode ser um caminho da da internet ou da pasta ou imagem puxada da internet, clico com o botão direito em cima da imagem e copiar o endereço da imagem e colar.

Textos

(voltar)

Texto é o que o usuário vai ler ou seja tipografia, na internet posso pesquisar como tipografia html. A tag mais simples são as h, tem h1, h2, h3 e vai até a h6, significa head, cabeçalho, título.

O corpo do texto é utilizado o p para parágrafos, a cada quebra de linha é usado um p novo. O navegador entende que o h1 é um título e o h2 é subtítulo e já separa com espaçamento, para alterar usamos o css.

Clicando o f12(inspetor de elemento), aparece a formatação para ver como está se comportando, ao passar o mouse em cima do título fica azul e laranja que significa margem e o próprio navegador define a margem.

Para aumentar e diminuir a tela, clico no segundo item, posso definir que tipo de dispositivo quero simular na tela, consigo ter uma prévia, isso é designer responsivo.

Quando tem uma citação, uso a Tag blockquote, ou usado para observação e abro um bloquinho e dar um espaçamento, se desprende do texto e posso formatar usando o css, posso colocar uma bordinha do lado ou aspas grandes.
